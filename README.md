# Лабораторная работа №2: Структуры в C (Вариант 13)

## Overview
Программа для работы с абонентами (структура ABON). Реализует:
- Создание массива абонентов (до 12 записей) с вводом данных с клавиатуры
- Сортировку записей по алфавиту (фамилия и инициалы)
- Добавление 20 рублей на счета абонентов, подключенных более 10 лет назад
- Вывод сведений об абонентах с отрицательным балансом после вычета начислений
- Поиск абонента по номеру телефона

Организация кода выполнена согласно модели КИС-2:
- **Клиент**: `main.c` — точка входа, вызов функций
- **Сервер**: `abonent.c`, `abonent.h` — логика обработки данных
- **Интерфейс**: `interface.c` — функции ввода/вывода
