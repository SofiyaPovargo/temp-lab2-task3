## Описание программы
Программа реализует работу с абонентами (структура ABON) и выполняет следующие функции:
1. Создание массива абонентов (до 12 записей)
2. Сортировка по алфавиту (фамилия и инициалы)
3. Добавление 20 руб. на счета абонентов, подключенных более 10 лет назад
4. Вывод абонентов с отрицательным балансом после вычета начислений
5. Поиск абонента по номеру телефона

## Структура проекта (КИС-2)
- **Клиент**: main.c - точка входа, вызов функций
- **Сервер**: abonent.c, abonent.h - логика обработки данных
- **Интерфейс**: interface.c - ввод/вывод данных
